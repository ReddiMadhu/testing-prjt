# Assets Directory

This folder contains static assets for the EXL FNOL Transcript Analyzer.

## Contents

- `exl_logo.png` - EXL company logo (add your logo file here)

## Usage

Place the EXL logo file named `exl_logo.png` in this directory.
The application will automatically use it in the header and sidebar.

## Recommended Specifications

- Format: PNG with transparent background
- Size: 200x60 pixels (or similar aspect ratio)
- Color: Orange (#E85D04) or white for dark backgrounds
